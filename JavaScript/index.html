what spread operator do ?

The spread operator (...) in JavaScript is used to expand (spread out) elements of an array, object, or iterable into individual elements.

const arr = [1, 2, 3]; const arr2 = [...arr];
    What happens?
  • ...arr takes each element from arr
  • Creates a new array
  • Copies values one by one
arr ‚Üí 1 2 3 ...arr ‚Üí 1 , 2 , 3 arr2 ‚Üí [1, 2, 3] (new memory)
